# 随机抽取工具

> 在线版：[https://lisrain.github.io/rdname/](https://lisrain.github.io/rdname/)

## 功能特性

- **号数/人名双模式** - 一键切换抽取号码或人名
- **隐蔽性排除** - 排除的项目正常显示但不参与抽取
- **响应式设计** - 支持手机、平板、桌面端自适应
- **全屏模式** - 适配教室大屏展示
- **动画效果** - 抽取时脉冲高亮，选中时弹跳放大
- **历史记录** - 自动记录已抽取的项目
- **本地存储** - 配置和历史记录保存在浏览器本地

## 快速开始

1. 访问 [在线版本](https://lisrain.github.io/rdname/)
2. 点击顶部「号数/人名」切换模式
3. 点击「开始抽取」按钮
4. 点击「停止」完成抽取

## 设置说明

点击右上角 ⚙ 按钮打开设置面板：

- **基本设置**: 调整号码总数、起始号码、动画速度、每次抽取数量
- **排除列表**: 勾选不参与抽取的项目（项目仍会显示）
- **人名管理**: 添加、删除、导入/导出人名列表

## 人名导入格式

支持 TXT 文件导入，每行一个名字：

```
张三
李四
王五
```

也支持逗号、分号、制表符分隔：

```
张三,李四,王五
```

## 快捷键

- `Space` / `Enter` - 开始/停止抽取
- `F` - 全屏模式
- `S` - 打开设置
- `Esc` - 关闭设置/停止抽取

## 技术栈

- 纯 HTML/CSS/JavaScript（无框架依赖）
- CSS Grid + Flexbox 响应式布局
- localStorage 本地持久化
- HarmonyOS Sans 字体
